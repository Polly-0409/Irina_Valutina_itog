# Ирина Валутина — лендинг косметолога

Лендинг для косметолога Ирины Валутиной (Кострома). Статический сайт без фреймворков и сборщиков — чистый HTML, CSS и JavaScript.

## Структура проекта

```
├── index.html          # Главная страница
├── style.css           # Стили
├── для постов.png      # Логотип (лайн-арт)
├── фото мастера.PNG    # Фотография специалиста
├── prompt.md           # Исходное ТЗ
└── SKILL.md            # Cursor skill (дизайн-гайд)
```

## Секции сайта

1. **Hero** — заголовок с blur-анимацией, логотип-иллюстрация, CTA
2. **Запросы** — карточки с типичными задачами клиентов
3. **Для кого** — нумерованный список целевой аудитории
4. **Консультация** — 5 шагов первой встречи
5. **Про специалиста** — фото, описание, ценности
6. **Методы** — три направления: лицо, тело, дополнительно
7. **Результаты** — карточки с ожидаемыми изменениями
8. **Сомнения** — опасения vs подход
9. **Формат работы** — офлайн, индивидуально, системно
10. **Отзывы** — перетаскиваемые карточки (drag-to-shuffle)
11. **FAQ** — аккордеон
12. **CTA** — финальный блок записи

## Технологии

- **HTML5** — семантическая разметка
- **CSS3** — CSS-переменные, Grid, Flexbox, анимации, `backdrop-filter`
- **JavaScript** — IntersectionObserver (scroll-анимации), drag-события (карточки отзывов), FAQ-аккордеон
- **Google Fonts** — Cormorant Garamond + DM Sans

## Запуск

Открыть `index.html` в браузере. Сборка не требуется.

Для локальной разработки с live-reload:

```bash
# Через npx (Node.js)
npx serve .

# Через Python
python3 -m http.server 8000
```

## Дизайн

- Палитра: тёплые тона — слоновая кость, антикварное золото, глубокий шоколад
- Типографика: Cormorant Garamond (заголовки) + DM Sans (текст)
- Анимации: scroll-reveal, blur-text, drag-карточки, hover-микроинтеракции
- Адаптивность: десктоп → планшет → мобильный
